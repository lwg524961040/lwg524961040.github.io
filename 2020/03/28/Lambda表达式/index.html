Lambda表达式

使用Lambda的原因

对接口进行简洁实现

new Thread(new Runnable() {
        @Override
        public void run() {
            System.out.println();
        }
    }).start();

使用Lambda

new Thread(()->{
       System.out.println();
   }).start();

Lambda表达式简介

java8新特性，说白了就是一个匿名函数
它只支持函数式接口
函数式接口:一个有且仅有一个抽象方法，但是可以有多个非抽象方法的接口。

() 用来描述参数列表 当接口只有一个参数时可以省略括号 参数类型与接口参数类型相同，且可以不写,即

 //无参
Interface1 i1 = () -> {
    System.out.println("!");
};
i1.noParameter();

 //单个参数
Interface2 i2 = i -> {
    System.out.println(i);
};
i2.oneParameter(2);

如果需要省略参数类型时，需要所有参数均进行省略

{} 用来描述方法体
当方法体中只有一条语句时，大括号也可以省略
Interface1 i1 = () -> System.out.println(“!”);
如果方法体重只有一条语句，且为返回语句，那么需要在省略大括号的同时 省略 return

interface Interface3{
    int returnFunction();
}

Interface3 i3 = () -> 10;

-> lambda运算符 读作 goes to

静态方法为类 非静态方法为对象

 public static int test(int i){
